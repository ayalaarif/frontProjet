À Propos

La plate-forme est créée pour fournir aux étudiants un accès à toutes les informations en relation avec l'université; telles que la description des UVs et les associations, en un seul endroit. Elle permet aux étudiants de trouver facilement les informations dont ils ont besoin. De plus, la plate-forme comporte les commentaires d'anciens étudiants de certaines UVs ou de membres d'une association pour que les étudiants puissent se faire une idée sur le sujet avant de faire leur choix.
